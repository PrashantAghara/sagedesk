# SageDesk 🧠

> RAG Knowledge Assistant + AI Customer Support Agent

An agentic AI system that answers questions from your documents
and handles customer support tickets — powered by LangGraph, Groq, and AstraDB.

## Stack
- **Orchestration** — LangGraph + LangChain
- **LLM** — Groq (llama-3 / llama-4)
- **Embeddings** — HuggingFace sentence-transformers
- **Vector Store** — AstraDB
- **Database** — PostgreSQL
- **UI** — Streamlit
- **API** — FastAPI
